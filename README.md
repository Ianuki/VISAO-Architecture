# VISAO Architecture

The VISAO CPU Architecture was created as a simple, fast and minimalistic 
CPU design for survival Minecraft Servers.

## Instruction set:

NOP   0000  -> REG
AND   0001  -> REG
OR    0010  -> REG
NAND  0011  -> REG
NOR   0100  -> REG
XOR   0101  -> REG
ADD   0110  -> REG
SL    0111  -> REG
STA   1000  -> VAL
LDA   1001  -> REG
STB   1010  -> VAL
LDB   1011  -> REG
